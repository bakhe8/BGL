# تجهيز نسخة سطح المكتب
خطوات إخراج نسخة جاهزة للمستخدم النهائي.

1) تنظيف المشروع: احذف ملفات التطوير المؤقتة وأبقِ `www/` و `storage/` الأساسية فقط.  
2) ضع ملفات الواجهة في `www/` الخاصة بـ PHP Desktop، وتأكد من مسارات الأصول.  
3) انسخ `storage/database/app.sqlite` فارغة أو بنسخة بيانات تجريبية حسب الحاجة.  
4) عدّل إعدادات `settings.json` في PHP Desktop لتعطيل DevTools إن لزم.  
5) اختبر الحزمة على جهاز نظيف (بدون أدوات تطوير مثبتة).  
6) اضغط المجلد كاملًا كأرشيف zip للتوزيع.
