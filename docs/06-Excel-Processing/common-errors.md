# أخطاء شائعة في الاستيراد
رسائل خطأ قياسية وكيفية تصحيحها.

- `MISSING_COLUMN`: أحد الأعمدة الإلزامية غير موجود. → أضف العمود وأعد الرفع.  
- `INVALID_DATE_FORMAT`: تاريخ لا يمكن تحويله. → استخدم YYYY-MM-DD.  
- `INVALID_AMOUNT`: المبلغ لا يمكن تحويله لرقم. → أزل الرموز وأعد المحاولة.  
- `EMPTY_ROWS`: تم الكشف عن صفوف فارغة كثيرة. → نظّف الملف لتقليل الحجم.  
- `UNEXPECTED_ENCODING`: الملف ليس UTF-8. → احفظه كـ UTF-8.

كل خطأ يُعرض مع رقم الصف لتسهيل التصحيح السريع.
