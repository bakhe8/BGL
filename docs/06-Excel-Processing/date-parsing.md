# تحليل التواريخ
كيف نتعامل مع صيغ التواريخ المختلفة.

- قبول صيغ: `YYYY-MM-DD`, `DD/MM/YYYY`, أرقام Excel التسلسلية.  
- عند استقبال رقم Excel: استخدم تحويل إلى تاريخ ميلادي ثم حفظ بصيغة `YYYY-MM-DD`.  
- إذا فشل التحويل: إرجاع خطأ `INVALID_DATE_FORMAT` مع رقم الصف.  
- تخزين التواريخ كنص ISO لتفادي اختلاف المناطق الزمنية.
