# الأرقام والمبالغ
تنظيف وتحويل قيم المبالغ.

- إزالة الفواصل/المسافات غير الرقمية (`1,200.50` → `1200.50`).  
- دعم الفاصلة العشرية العربية (`1.200,50` → `1200.50`).  
- رفض القيم السالبة أو النصوص غير الرقمية.  
- حفظ المبلغ كقيمة عددية + نسخة نصية خام لأغراض التدقيق.  
- عملة افتراضية يمكن تحديدها في الإعدادات، أو استخدم عمود `currency` إن وجد.
