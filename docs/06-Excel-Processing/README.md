# 06-Excel-Processing — معالجة الإكسل
قواعد قراءة ملفات Excel، التحقق من الصفوف، ومعالجة التواريخ والأرقام قبل المطابقة.

## المحتويات
- **excel-format.md** — الأعمدة المطلوبة، الترتيب، وأي تنسيقات مقبولة.
- **validation-rules.md** — ما الذي نرفضه/نقبله عند الاستيراد.
- **date-parsing.md** — تحويل صيغ التواريخ المختلفة وضبط التوقيت.
- **numbers-and-amounts.md** — تنظيف أرقام المبالغ والأصفار والفواصل.
- **common-errors.md** — أخطاء شائعة وكيف تُعرض للمستخدم.
