# قواعد التحقق
الفحوصات التي تتم قبل قبول الصف.

- **الأعمدة الإلزامية** موجودة وغير فارغة.  
- **تواريخ صالحة**: يمكن تحويلها إلى تاريخ ISO؛ رفض القيم غير القابلة للتحويل.  
- **المبلغ**: رقم موجب، يسمح بالفواصل والنقاط؛ يتم تحويله إلى قيمة رقمية.  
- **نصوص قصيرة**: لا تتجاوز 255 حرفًا للأسماء/الأرقام.  
- **ترميز**: يتم تحويل النصوص إلى UTF-8 قبل المعالجة.

عند فشل أي قاعدة، يتم إرجاع خطأ واضح مع رقم الصف والمشكلة.
