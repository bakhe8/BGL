# مبدأ الترقيات (Migrations)
كيفية إضافة/تعديل الجداول بأمان.

1) أنشئ ملف SQL جديد في `database/migrations/` باسم تاريخي واضح:  
   `YYYY-MM-DD-short-title.sql`.  
2) اجعل التغييرات **إضافية** قدر الإمكان (إضافة جداول/حقول).  
3) شغّل الملف على نسخة التطوير:  
   ```
   sqlite3 storage/database/app.sqlite ".read database/migrations/<file>.sql"
   ```  
4) حدث `schema.sql` لتعكس البنية النهائية.  
5) زِد `schema_meta` إن وُجدت نسخة.  
6) احتفظ بنسخة احتياطية قبل أي تعديل في بيئة المستخدم.

لا تحذف الأعمدة مباشرة؛ استخدم جداول بديلة وانسخ البيانات إذا احتجت تغييرًا جذريًا.
