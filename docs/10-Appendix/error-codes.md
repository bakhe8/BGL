# أكواد الأخطاء الموحدة
استخدم هذه الأكواد في الاستجابات واللوج.

- `IMPORT_MISSING_COLUMN` — عمود أساسي غير موجود.  
- `IMPORT_INVALID_DATE` — قيمة تاريخ غير صالحة.  
- `IMPORT_INVALID_AMOUNT` — المبلغ غير قابل للتحويل.  
- `MATCH_NO_RESULT` — لم يتم إيجاد تطابق مناسب.  
- `MATCH_OVERRIDE_REQUIRED` — يحتاج Override صريح.  
- `DB_LOCKED` — تعارض في قفل SQLite.  
- `UNEXPECTED_ERROR` — خطأ غير متوقع (يجب تسجيل الـ trace).
