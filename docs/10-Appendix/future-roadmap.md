# خارطة الطريق المقترحة
- إضافة طبقة مستخدمين وصلاحيات بسيطة (قراءة/كتابة/إدارة قاموس).  
- دعم استيراد CSV مباشر مع كشف الأعمدة تلقائيًا.  
- تحسين محرك المطابقة بإضافة نموذج تشابه متقدم أو مكتبة صوتية للأسماء العربية.  
- واجهة تقارير لتلخيص نسب الاعتماد/المراجعة.  
- أداة مزامنة اختيارية مع خادم مركزي (اختياري ومشفر).  
- حزمة اختبارات تلقائية CI محلية.
