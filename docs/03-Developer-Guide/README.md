# 03-Developer-Guide — دليل المطور
كل ما يحتاجه المطور لفهم المعمارية، المعايير، وتدفقات العمل في الباك‑إند والفرونت.

## المحتويات
- **project-architecture.md** — خريطة الطبقات (Desktop shell، PHP، SQLite، الواجهة).
- **folder-structure.md** — بنية الملفات المتوقعة داخل المشروع.
- **coding-standards.md** — معايير التنسيق والتسمية.
- **php-backend-flow.md** — تدفق الطلبات في PHP (مسارات داخل index.php → Controllers → Services → Repositories).
- **js-frontend-flow.md** — تدفق الواجهة (الإشعارات، الطلبات، التحديثات).
- **naming-conventions.md** — ملخص سريع للقواعد المعتمدة (يرتبط بالملف العام).
- **adding-new-features.md** — خطوات إضافة ميزة جديدة دون كسر النظام.
- **troubleshooting.md** — أعطال شائعة وكيفية تشخيصها.

اقرأ **project-architecture.md** أولًا ثم **folder-structure.md**، وبعدها انتقل للتدفقات حسب الحاجة.
