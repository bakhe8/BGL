# المكونات الأساسية
مكتبة صغيرة من المكونات القابلة لإعادة الاستخدام.

- **Primary Button**: لون أساسي، حواف ناعمة، حالة hover/disabled واضحة.  
- **Secondary Button**: حواف خطية، للخيار الثانوي.  
- **Badge/Chip**: لعرض حالة المطابقة (`auto`, `needs_review`, `manual`).  
- **Table**: صفوف متباعدة، أشرطة بديلة للألوان، رموز حالة في العمود الأول.  
- **Drawer/Modal**: لعرض تفاصيل السجل أو إنشاء مورد جديد.  
- **Inline Alert**: أخضر للنجاح، أصفر للتحذير، أحمر للخطأ.  
- **Tabs**: لتبديل المشاهد (بدائل/Overrides/سجل التدقيق) في لوحة المراجعة.

استخدم نفس المكونات في كل الشاشات لتجنب تشتت بصري.
