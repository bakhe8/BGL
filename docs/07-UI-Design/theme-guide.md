# دليل الثيم
الألوان والخطوط والحالات الأساسية.

- **الألوان**  
  - Primary: Blue 600 (#2563eb)  
  - Secondary/Neutral: Slate 700/600  
  - Success: Green 600  
  - Warning: Amber 500  
  - Danger: Red 600  
  - Background: Slate 50

- **الخط**  
  - خط عربي/لاتيني متوافق مثل "Cairo, Segoe UI, sans-serif".  
  - حجم أساسي 14px، line-height 1.5.

- **الحالات**  
  - Hover: تفتيح اللون أو إضافة ظل خفيف.  
  - Disabled: تقليل التباين (`opacity-50`, `cursor-not-allowed`).  
  - Focus: حدود زرقاء رفيعة `ring-2 ring-blue-200`.

حافظ على تباين كافٍ للقراءة واستخدم نفس لوحة الألوان في كل الشاشات.
