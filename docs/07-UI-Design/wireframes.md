# تخطيطات مبدئية
وصف سريع للصفحات الرئيسية (قابل للتفصيل لاحقًا).

- **شاشة السجلات (Import/Review)**  
  - شريط علوي يحوي أزرار الاستيراد (Excel / لصق / يدوي).  
  - جدول سجلات مع أعمدة الحالة، المورد المقترح، البنك، المبلغ، التواريخ.  
  - عند الضغط على صف، يفتح Drawer يمين الشاشة بتفاصيل السجل والقرارات.

- **شاشة القاموس**  
  - تبويب للموردين وآخر للبنوك.  
  - جدول بأسماء رسمية + أزرار تعديل/حذف، مع عداد للأسماء البديلة.  
  - نموذج جانبي لإضافة مورد/بنك سريعًا.

- **شاشة التصدير**  
  - اختيار نوع التصدير ونطاق الجلسات.  
  - قائمة بالملفات المصدرة سابقًا مع إمكانية فتحها في Explorer.
