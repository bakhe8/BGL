# 07-UI-Design — تصميم الواجهة
دليل تخطيط الشاشات والمكونات في نسخة PHP Desktop (HTML/CSS/JS).

## المحتويات
- **layout-rules.md** — نظام الشبكة، الهوامش، وأنماط الصفحات.
- **components.md** — المكونات الأساسية (جداول، نماذج، إشعارات).
- **tailwind-classes.md** — فئات Tailwind المعتمدة مسبقًا.
- **theme-guide.md** — الألوان، الخطوط، والحالات (فعال/خطأ/تحذير).
- **wireframes.md** — تخطيطات سريعة للصفحات الرئيسية.
