# فئات Tailwind المعتمدة
مجموعة أساسية لتقليل العشوائية في الأنماط.

- **نص/خط**: `text-sm`, `text-base`, `text-lg`, `font-medium`, `font-semibold`, `leading-5`.  
- **ألوان أساسية**:  
  - خلفية: `bg-slate-50`, `bg-white`  
  - نص: `text-slate-800`, `text-slate-600`  
  - أساس: `bg-blue-600 text-white` للأزرار الأساسية  
  - تحذير/خطأ: `bg-amber-100 text-amber-900`, `bg-red-100 text-red-900`
- **مسافات**: `p-4`, `p-6`, `gap-3`, `gap-4`, `mt-2`, `mb-4`.  
- **حدود/ظلال**: `border`, `border-slate-200`, `rounded-md`, `shadow-sm`.  
- **جداول**: `divide-y divide-slate-200`, `whitespace-nowrap`, `text-left`.

أي فئة إضافية يجب أن تُضاف للقائمة بعد مراجعة التصميم لضمان الاتساق.
