# قواعد التخطيط
مبادئ الواجهة في نسخة سطح المكتب.

- شبكة 12 عمودًا بسيطة مع max-width 1200px في الشاشات الكبيرة.  
- هوامش داخلية 16px للمكونات، 24px للحاويات الكبيرة.  
- استخدم شريط علوي ثابت لحالة الاتصال والأزرار الأساسية.  
- أقسام الشاشة الرئيسية:
  - لوحة جانبية للتنقل (Import, Review, Dictionary, Export).
  - مساحة محتوى بعرض متبقٍ للجداول/النماذج.
- جدول المراجعة بعرض كامل مع صفوف قابلة للطي لعرض التفاصيل.
