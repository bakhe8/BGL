# 01-Setup

- install-project.md — إعداد وتشغيل المشروع
- configure-tailwind.md — تجهيز TailwindCSS محليًا
- environment-structure.md — بنية البيئة الخاصة بالمشروع
- dependencies.md — التبعيات البرمجية المطلوبة
