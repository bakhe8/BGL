# التبعيات Dependencies

## 1. PHP
- PHP 8.2 أو أحدث
- sqlite3
- pdo_sqlite
- mbstring
- json

## 2. Node (اختياري — لبناء Tailwind فقط)
- Node.js 18+
- tailwindcss
- postcss
- autoprefixer

## 3. PHP Desktop
- phpdesktop-chrome أو أي نسخة مناسبة

## 4. مكتبات PHP المقترحة
- phpoffice/phpspreadsheet لتحليل ملفات Excel
- vlucas/phpdotenv لإدارة الإعدادات
- symfony/routing (اختياري)
- symfony/http-foundation (اختياري)
