# اختبارات التكامل
تدفقات كاملة من الاستيراد حتى التصدير.

- **Import → Review → Export**: استيراد ملف، اعتماد النتائج، تصدير ملف نظيف، التأكد من القيم.  
- **Import with Errors**: ملف يحتوي تواريخ/مبالغ خاطئة → التحقق من رسائل الأخطاء وعدم إنشاء جلسة تالفة.  
- **Dictionary Update Flow**: إنشاء مورد جديد، ربط اسم بديل به عبر المراجعة، استيراد ملف آخر يحتوي الاسم البديل → يجب أن يطابق تلقائيًا.  
- **Override Flow**: إضافة Override ثم استيراد اسم مطابق → يتم اعتماده دون طلب مراجعة.  
- **Concurrency**: عمليات متتالية (استيراد + مراجعة) دون تعارض في قفل SQLite.
