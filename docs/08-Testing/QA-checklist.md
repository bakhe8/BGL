# قائمة فحص الجودة
افحص هذه البنود قبل أي تسليم.

- الاستيراد يقبل ملف نموذج ويولد جلسة بدون أخطاء.  
- لوحة المراجعة تعرض جميع الحالات وتسمح بالحفظ.  
- إنشاء مورد جديد ينعكس فورًا في شاشة القاموس والمراجعة.  
- التصدير ينتج ملفًا صالحًا ويفتح بدون تحذيرات.  
- السجلات في `logs/app.log` خالية من أخطاء غير متوقعة.  
- التطبيق يعمل دون اتصال بالشبكة.
