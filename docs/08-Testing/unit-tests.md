# اختبارات الوحدات
ما يجب تغطيته في المنطق الداخلي.

- دوال التطبيع (أحرف عربية/إنجليزية، إزالة الرموز).  
- حساب درجات الثقة والعتبات.  
- خوارزمية المطابقة (لكل مصدر: override, official, alt_confirmed, fuzzy).  
- تحديث الأسماء البديلة (`occurrence_count`, `last_seen_at`).  
- التحقق من صحة المدخلات (Validator) مع حالات النجاح والفشل.

استخدم SQLite في وضع in-memory للاختبارات السريعة مع بيانات عينات.
