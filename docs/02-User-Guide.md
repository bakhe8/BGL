# 02 - دليل المستخدم (User Guide)

## 1. استيراد ملف جديد
1. اذهب إلى صفحة **السجلات**.
2. اضغط على زر **استيراد ملف Excel**.
3. اختر الملف من جهازك.
4. سيقوم النظام بمعالجة الملف وعرض النتائج:
   - **اللون الأخضر**: تم الاستيراد بنجاح.
   - **اللون الأحمر**: تم تخطي السجلات (مع ذكر السبب، مثل تكرار رقم الضمان).

> **ملاحظة**: إذا قمت باستيراد رقم ضمان موجود مسبقاً، سيتم قبوله كسجل جديد بتاريخ اليوم ("سجل تاريخي").

## 2. مراجعة السجلات
بعد الاستيراد، ستظهر السجلات في الجدول بالحالات التالية:
- **جاهز (Ready)**: النظام تعرف على المورد والبنك بنسبة ثقة عالية.
- **يحتاج مراجعة (Needs Review)**: النظام غير متأكد أو المعلومات ناقصة.

### اتخاذ القرار
عند فتح أي سجل، يمكنك:
- **اعتماد الاقتراح**: إذا كان الاقتراح صحيحاً.
- **البحث اليدوي**: الكتابة في خانة البحث لاختيار مورد آخر.
- **إضافة اسم جديد**: إذا كان المورد غير موجود في القائمة، يمكنك إضافته وسيتعلمه النظام.

## 3. الإعدادات
من صفحة الإعدادات، يمكنك التحكم في:
- **الأوزان (Weights)**: أهمية المطابقة الرسمية مقابل التقريبية.
- **العتبات (Thresholds)**: النسبة المئوية المطلوبة للقبول التلقائي.
- **عدد النتائج**: عدد الاقتراحات التي تظهر في القائمة.
