# Overrides
آلية فرض اسم معيّن عند المطابقة في حالات خاصة.

## الاستخدامات المسموح بها
- تصحيح أسماء تتكرر بخطأ مطبعي ثابت لا يمكن معالجته بالتطبيع.
- حالات قانونية/مالية تتطلب اسمًا محددًا بغض النظر عن النتيجة.

## كيفية الإضافة
1) من لوحة المراجعة أو شاشة القاموس اختر السجل ثم **إضافة Override**.  
2) أدخل `override_name` مع ملاحظة سبب الاستخدام.  
3) الحفظ ينشئ سجلًا في `supplier_overrides`.

## تأثيرها
- في المحرك: يتم فحص Overrides قبل الأسماء البديلة.  
- تظهر للواجهة كـ **قيمة مفروضة** ويجب إقرارها بوضوح.

## ضوابط
- لا تستخدم Override إذا كان التطبيع أو الأسماء البديلة كافيًا.  
- راجع Overrides دوريًا لإزالة غير الضروري.
