# الأسماء البديلة
طريقة توثيق أسماء الموردين المختلفة لرفع دقة المطابقة.

## متى نستخدمها؟
- عند ظهور اسم خام مختلف لنفس المورد (خلال المراجعة).
- عند إضافة مورد جديد وتريد تسجيل الأسماء الشائعة له.
- بعد تصحيح خطأ مطابقة متكرر.

## الإجراء
1) من لوحة المراجعة، بعد اختيار المورد الصحيح، فعّل **أضف كاسم بديل**.  
2) النظام يسجل `raw_name` و`normalized_raw_name` مع `supplier_id` ومصدره.  
3) تزداد `occurrence_count` تلقائيًا عند تكرار الاسم.

## مبادئ
- لا تضف أسماء عامة جدًا (مثل "مورد متنوع").  
- احتفظ بالاسم الخام كما جاء في الإدخال، ولا تفرض تنسيقًا جديدًا.  
- يمكن إدارة الأسماء البديلة لاحقًا من شاشة القاموس.
