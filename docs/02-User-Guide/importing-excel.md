# استيراد ملفات Excel
خطوات تحميل ملف وضبطه قبل الدخول لمرحلة المراجعة.

## قبل الرفع
- تأكد من الأعمدة الأساسية: `supplier_name`, `bank_name`, `amount`, `guarantee_number`, `issue_date`, `expiry_date`.
- أزل الصفوف الفارغة أو المكررة بوضوح.
- احفظ الملف بصيغة `.xlsx` (بدون صيغ معقدة أو ماكرو).

## الرفع
1) من شاشة السجلات اضغط **استيراد Excel**.  
2) اختر الملف واضغط **Upload**.  
3) النظام يحلل الأعمدة ويعرض ملخص عدد الصفوف المقبولة/المرفوضة.  

## ما يحدث بعد الرفع
- يتم إنشاء جلسة استيراد في `import_sessions`.
- كل صف يمر على التطبيع والمطابقة الأولية.
- النتيجة تنتقل مباشرة لشاشة المراجعة.

## أخطاء شائعة
- **عمود ناقص**: أضف العمود ثم أعد الرفع.  
- **صيغة تاريخ غير مفهومة**: صححها أو استخدم تنسيق YYYY-MM-DD.  
- **ملف كبير جدًا**: قسم الملف لدفعات أصغر.
