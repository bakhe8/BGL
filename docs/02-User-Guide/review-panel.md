# لوحة المراجعة
المكان الذي تُراجع فيه نتائج المطابقة وتُتخذ القرارات النهائية.

## عناصر الواجهة
- جدول السجلات مع حالة كل صف: `auto`, `needs_review`, `manual`.
- بطاقة تفاصيل السجل (الاسم الخام، الاسم المقترح، البنك، المبلغ، التواريخ).
- قائمة اقتراحات الموردين مع درجة الثقة.
- أزرار القرار: **اعتماد**، **رفض**، **تعديل يدوي**، **إضافة اسم بديل**.

## طريقة العمل
1) راجع الاسم المقترح ودرجة الثقة.  
2) إذا صحيح → اضغط **اعتماد** (يُسجل القرار).  
3) إذا خطأ → اختر المورد الصحيح أو اكتب اسمًا جديدًا ثم **اعتماد**.  
4) إذا الاسم الجديد يجب حفظه كبديل → فعّل خيار **أضف كاسم بديل**.  
5) في الحالات غير الواضحة → اتركه **needs_review** لتعود له لاحقًا.

## نصائح
- قراراتك تُحسّن المطابقة لاحقًا عبر جدول الأسماء البديلة.  
- يمكنك إضافة مورد جديد مباشرة دون مغادرة الشاشة.  
- سجلات القرار تُحفظ في سجل تدقيق للتتبع.
