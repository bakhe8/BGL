# الإدخال عبر اللصق
للبيانات السريعة بدون رفع ملف.

## طريقة الاستخدام
1) انسخ الصفوف من Excel أو نص منسّق بفواصل أو تبويب.  
2) افتح **إدخال باللصق**، وألصق النص في الحقل.  
3) اضغط **تحليل** ليتعرف النظام على الأعمدة.  
4) صحح أي عمود غير معرّف ثم اضغط **استيراد**.

## ملاحظات
- احرص على وجود نفس الأعمدة المطلوبة في الاستيراد العادي.  
- يتم إنشاء جلسة استيراد بنفس طريقة ملفات Excel.
