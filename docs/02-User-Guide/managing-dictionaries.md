# إدارة القواميس
إدارة الموردين، البنوك، والأسماء المرتبطة بهم.

## الموردون (Suppliers)
- إنشاء: اسم رسمي + اسم عرض عربي اختياري.
- تعديل: تحديث الاسم الرسمي أو علامة التأكيد `is_confirmed`.
- حذف: لا ينصح بحذف المورد المرتبط بسجلات؛ استخدم تعطيل منطقيًا إذا لزم.

## البنوك (Banks)
- نفس مبادئ الموردين: اسم رسمي، اسم عرض، وتأكيد.

## الأسماء البديلة
- تعرض الجدول مع `occurrence_count` و`last_seen_at`.
- يمكن ربط بديل بمورد آخر عند اكتشاف تطابق خاطئ.

## البحث والتصفية
- بحث بالحروف العربية/الإنجليزية بعد التطبيع.
- فلاتر حسب حالة التأكيد أو عدد التكرار.

## نصائح
- حدّث القاموس بعد كل دفعة استيراد كبيرة.  
- راجع التكرارات المرتفعة لتقليل أخطاء المستقبل.
