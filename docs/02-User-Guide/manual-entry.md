# الإدخال اليدوي
إضافة سجل واحد دون استيراد ملف كامل.

## الخطوات
1) من شاشة السجلات اختر **إضافة يدويًا**.  
2) أدخل الحقول الأساسية:
   - اسم المورد (إلزامي)
   - اسم البنك
   - رقم الضمان
   - المبلغ
   - تاريخ الإصدار / الانتهاء  
3) اضغط **حفظ** للانتقال مباشرة للمراجعة.

## قواعد سريعة
- الحقول الإلزامية يجب أن تكون نصًا واضحًا بدون رموز غريبة.  
- التاريخ يُفضَّل بصيغة YYYY-MM-DD.  
- كل إدخال يدوي يُنشئ جلسة استيراد صغيرة لتتبع المصدر.
