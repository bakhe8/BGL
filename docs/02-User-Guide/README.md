# 02-User-Guide — دليل الاستخدام
إرشادات المستخدم لتشغيل النظام اليومي: الاستيراد، المراجعة، إدارة القواميس، والتصدير.

## المحتويات
- **importing-excel.md** — خطوات تحميل ملفات Excel وما يجب التحقق منه قبل الرفع.
- **paste-input.md** — إدخال سريع عبر اللصق.
- **manual-entry.md** — إضافة سجل واحد يدويًا.
- **review-panel.md** — كيفية اتخاذ قرارات المطابقة والتعامل مع الحالات الشاذة.
- **alternative-names.md** — إضافة/إدارة الأسماء البديلة أثناء المراجعة.
- **overrides.md** — متى تستخدم Overrides وكيف تؤثر على المطابقة.
- **managing-dictionaries.md** — CRUD على الموردين والبنوك.
- **exporting-data.md** — طرق التصدير (سجلات نظيفة، قواميس، سجلات تدقيق).

ابدأ بملفي **importing-excel.md** و **review-panel.md**، ثم انتقل لباقي المواضيع حسب الحاجة.
