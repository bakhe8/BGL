# تصدير البيانات
خيارات استخراج النتائج بعد المراجعة.

## أنواع التصدير
- **سجلات نظيفة**: ملف Excel جديد يحتوي على الأسماء المعتمدة والمعلومات المصححة.
- **قاموس الموردين**: CSV/JSON للأسماء الرسمية والبديلة.
- **سجل التدقيق**: JSON بسجل القرارات لتتبع ما حدث في كل خطوة.

## خطوات التصدير
1) افتح شاشة التصدير واختر نوع التصدير.  
2) حدد نطاق السجلات (كل الجلسات أو جلسة محددة).  
3) اضغط **Export** واحفظ الملف محليًا.

## ملاحظات
- التأكد من انتهاء حالة **needs_review** قبل التصدير النهائي.  
- يحفظ النظام نسخة في مجلد `exports/` للرجوع إليها.  
- يمكن استخدام ملفات القاموس للتدريب أو الدمج في نسخ أخرى.
