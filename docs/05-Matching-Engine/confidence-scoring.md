# درجات الثقة
كيفية حساب العتبات واتخاذ القرار التلقائي.

- `override_name` → درجة 1.0 (لا يحتاج مراجعة).  
- تطابق رسمي كامل → 0.95+ (غالبًا اعتماد تلقائي).  
- اسم بديل مؤكد → 0.85.  
- اسم بديل غير مؤكد/تعلم → 0.75.  
- تطابق تقريبي خام → 0.60 (يحتاج مراجعة).

## العتبات
- `MATCH_AUTO_THRESHOLD` = 0.90 → اعتماد تلقائي.  
- `MATCH_REVIEW_THRESHOLD` = 0.70 → عرض للمراجعة مع تمييز تحذيري.  
- أقل من 0.70 → لا اقتراح، يطلب اختيار يدوي.

سجل الدرجة ومصدرها في `learning_log` لمراقبة التحسينات.
