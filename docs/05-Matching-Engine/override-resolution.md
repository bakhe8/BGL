# حل الـ Overrides
قواعد أولوية الأسماء المفروضة.

- يتم فحص جدول `supplier_overrides` أولًا بعد التطبيع مباشرة.  
- إذا وُجدت قيمة مطابقة، يتم إرجاع المورد المرتبط بدون حسابات إضافية.  
- يجب إرفاق `notes` لتوضيح سبب الإضافة.  
- Overrides لا تنشئ أسماء بديلة تلقائيًا؛ هي حكم صريح فقط.  
- يوصى بمراجعة القائمة دوريًا وإزالة ما لم يعد ضروريًا.
