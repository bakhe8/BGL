# تدفق الأسماء البديلة
كيف تُنشأ وتُحدّث الأسماء البديلة في النظام.

1) **الاستيراد**: أي اسم خام يتم تطبيعه؛ إذا لم يوجد في القاموس يحفظ كمرشح تعلم (بدون تأكيد).  
2) **المراجعة**: عندما يختار المستخدم المورد الصحيح ويفعل خيار "أضف كاسم بديل" يُنشأ سجل مؤكد.  
3) **الاستخدام**: كل مرة يتم اختيار الاسم يزداد `occurrence_count` ويتم تحديث `last_seen_at`.  
4) **التنظيف**: الأسماء ذات التكرار المنخفض وبدون تأكيد يمكن مراجعتها أو حذفها بعد فترة.

المصدر (`source`) يجب أن يحدد: `import`, `manual`, `paste`, `override`, `learning`.
