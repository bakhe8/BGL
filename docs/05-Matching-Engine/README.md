# 05-Matching-Engine — محرك المطابقة
القواعد المنطقية لتطبيع الأسماء، البحث، حساب الثقة، والتكامل مع شاشة المراجعة.

## المحتويات
- **normalization.md** — قواعد تنظيف النص (عربي/إنجليزي، مسافات، رموز).
- **matching-logic.md** — ترتيب طبقات المطابقة وقواعد الاختيار.
- **confidence-scoring.md** — كيفية حساب درجات الثقة والعتبات.
- **alternative-names-flow.md** — دورة حياة الأسماء البديلة والتعلم.
- **override-resolution.md** — أولوية Overrides وتأثيرها على النتائج.
- **review-panel-integration.md** — ما يتم تمريره للواجهة وكيف تُعرض النتائج.
