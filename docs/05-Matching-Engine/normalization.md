# التطبيع (Normalization)
قواعد تنظيف النص قبل المطابقة.

- إزالة المسافات الزائدة وتوحيد المسافات إلى فراغ واحد.  
- تحويل الحروف العربية الشائعة: (أ/إ/آ → ا)، (ة → ه عند الحاجة)، (ى → ي).  
- إزالة التشكيل، الرموز، والأرقام الزائدة بنهاية الاسم.  
- خفض الأحرف الإنجليزية إلى lowercase.  
- Unicode normalization (NFKC) لتوحيد الأشكال.  
- حفظ النسخة الخام دائمًا مع النسخة المطبعّة للمراجعة.
